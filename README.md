# Emat.dk
Hacks for Emat.dk a danish mathematics site

You have to use a javascript console to all scripts.

#-------Abbreviations/Forkortelser-------#

PC = PerClick

CPS = ClicksPerSecond

PS = PerSecond

$ = Dollar/money

#-------Examples/Eksempler-------#
Fx. 

3PC = 3 <value:Fx. answers> PerClick

2$PS = 2 dollars PerSecond
